Conway's Game of Life in J
======

(Inspired by: http://www.youtube.com/watch?v=a9xAKttWgP4)

Here's how to start the animation:

   * load'conway/conway.ijs'
   * conwaystart 100

Or faster:
   * 10 conwaystart 100

And to stop:
   * conwaystop''
